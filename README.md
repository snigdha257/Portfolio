## рџ“¦ Installation & Setup

### Prerequisites
- Node.js (v14+)
- npm or yarn
- Git

### Clone the Repository
```bash
git clone https://github.com/yourusername/portfolio.git
cd portfolio
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```

Frontend runs on: `http://localhost:3000`

## вљ™пёЏ Configuration

### Update Personal Information

1. **Hero Section** - Edit `frontend/src/components/Hero.js`
2. **About Section** - Edit `frontend/src/components/About.js`
3. **Projects** - Edit `frontend/src/components/Projects.js`
4. **Skills** - Edit `frontend/src/components/Skills.js`
5. **Internships** - Edit `frontend/src/components/Internships.js`
6. **Certifications** - Edit `frontend/src/components/Certifications.js`
7. **Contact** - Edit `frontend/src/components/Contact.js`

### Change Colors

Edit color variables in `frontend/src/index.css`:

```css
:root {
  --primary-color: #1a73e8;
  --secondary-color: #34a853;
  --accent-color: #fbbc04;
}
```
